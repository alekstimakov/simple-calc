Краткое описание

Это простой калькулятор, который принимает строку с арифметическим выражением и возвращает результат вычисления.

Он поддерживает числа с точкой, операции сложения, вычитания, умножения и деления, а также круглые скобки с правильным учётом приоритета операций.

Выражение обрабатывается без использования eval — строка сначала разбирается на числа, операторы и скобки, после чего вычисляется с помощью стеков.

Если выражение составлено неправильно, содержит недопустимые символы, лишние скобки или происходит деление на ноль, функция возвращает None.
